在 APP 开发过程中，很多问题是在网络问题出现的时候暴露出来的，所以在上线前，很有必要进行弱网测试。

# 工具：Charles

  1. 手机或模拟器设置代理

  2. Charles 开启代理功能

    勾选上 macOS 代理

  3. 进入【代理】- 【节流阀设置】 也叫 Throttle Setting


    ①勾选启用节流
    ②可以选择针对特定的 host 进行节流
    ③增加预设，这里我增加了一个弱网预设项，上传 30，下 行 15（这里我写反了）
    ④最后确定保存就好了

  4. 最后开始调试 APP 吧

  5. 后面我们可以直接通过开关启用关闭节流功能
