首页 Android

在更新 firebase services 的时候出现 connection Reset

  1. 手动下载 plugin 的 jar 包
    地址为 http://plugins.jetbrains.com/pluginManager/?action=download&id=com.google.services.firebase&build=AI-193.6911.18&uuid=9961ed02-3e65-41ef-9675-5d2fe4de63f2
    这里直接下载的话会失败，需要通过 https 下载，附上新的下载地址
    无法访问的话可以试试蓝奏云
  2. 将下载的压缩文件解压
  3. 最后执行本地安装

    选择 install plugin from disk
    重启 idea 后，大功告成
